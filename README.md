# Text Completion App

## How to set up?
1. npm install react-native-web
2. npm i semantic-ui-react
3. npm i semantic-ui-css 
4. npm install expo-status-bar
5. npm i openai
6. Rename apiKey in App.js with 'your personal apiKey'
7. cd gpt3-learn
8. npm start
